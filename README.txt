# المحلل الطبي الذكي

تطبيق ويب مبني باستخدام Streamlit لتحليل التقارير الطبية (صور) وتقديم استشارة أولية بناءً على الأعراض باستخدام الذكاء الاصطناعي.

## الميزات
- **تحليل التقارير:** رفع صورة لتقرير طبي واستخراج النتائج وتحليلها.
- **استشارة بالأعراض:** إدخال الأعراض والحصول على تحليل أولي ونصائح من نموذج GPT.

## التشغيل على Streamlit Cloud
تم تصميم هذا المشروع ليعمل مباشرة عند رفعه إلى GitHub وربطه بـ Streamlit Cloud. ملف `packages.txt` سيضمن تثبيت Tesseract OCR واللغة العربية تلقائيًا.
