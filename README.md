# 🩺 AI Medical Analyzer

## 🇸🇦 المميزات:
- 🧠 تحليل الأعراض عبر نموذج مدرب وذكاء اصطناعي.

- 🩻 تحليل فحوصات الدم والبول والبراز تلقائيًا.  
- 🗂️ واجهة تفاعلية باستخدام Streamlit.
